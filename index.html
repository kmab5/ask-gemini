Getting Started Guide

Set up Ask Gemini in under 2 minutes.

1

Get a Free API Key

Visit Google AI Studio and sign in with your Google account. Click "Create API Key" ‚Äî it's completely free!

2

Install the Extension

Add Ask Gemini to Chrome from the Chrome Web Store. Click the extension icon in your toolbar to open settings.

3

Paste Your API Key

Copy your API key from Google AI Studio and paste it into the extension settings. Choose "Simple Storage" for easy access, or "Password Protected" for extra security.

4

Start Using It!

Highlight any text on any webpage. A small Gemini button will appear ‚Äî click it to get an instant AI explanation. That's it!

Transparency

Why we need all-sites access

We believe in full transparency about permissions.

Single Purpose Statement

Ask Gemini has one single purpose: to explain highlighted text on any webpage using Google's Gemini AI.

To fulfill this purpose, the extension needs to detect when you select text on any website. This requires the <all_urls> permission.

What We Access

  • Text selection only ‚Äî Only the text you highlight is captured when you click our button
  • Nothing else ‚Äî We don't read page content, track browsing, or access any other data

Minimal Permission Use

  • We use activeTab permission where possible
  • Content scripts only inject minimal UI elements (a button and tooltip)
  • No background data collection or persistent tracking
  • Extension only activates when you select text
Legal

Privacy Policy

Overview

Ask Gemini is committed to protecting your privacy. This policy explains what data we collect, how we use it, and your rights.

Data Collection

Ask Gemini stores the following data locally on your device:

  • API Key: Your Google Gemini API key, stored in Chrome's sync storage or encrypted
  • Preferences: Your custom question template settings

We do NOT collect: Personal information, browsing history, analytics, usage data, or any data on external servers.

Data Usage

When you highlight text and click Ask Gemini:

  • The selected text is sent directly to Google's Gemini API using your API key
  • The request goes from your browser to Google's servers
  • No data passes through any servers owned by us

Third-Party Services

This extension uses the Google Gemini API. Please review Google's Privacy Policy for information on how they handle data.

Data Security

  • Your API key is stored in Chrome's built-in secure storage system
  • Optional password encryption uses AES-256-GCM with PBKDF2 key derivation
  • All API requests are made over HTTPS

Your Rights

  • Delete your stored API key and settings at any time through the extension popup
  • Uninstall the extension to remove all locally stored data
  • Clear your API key from Chrome's storage via Chrome settings

Contact

For questions about this privacy policy or the extension, please open an issue on our GitHub repository.
