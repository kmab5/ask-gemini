Why All-Sites Access?

Single Purpose Statement

Ask Gemini has one single purpose: to explain highlighted text on any webpage using Google's Gemini AI.

To fulfill this purpose, the extension needs to detect when you select text on any website you visit. This is why we request access to all URLs (<all_urls>).

What We Access

  • Text selection: Only the text you highlight is captured when you click our button.
  • Nothing else: We don't read page content, track your browsing, or access any other data.

Why Not Limit to Specific Sites?

Users want to look up definitions, explanations, and meanings while reading content across the entire web‚Äînews articles, documentation, social media, research papers, and more. Limiting the extension to specific sites would defeat its core purpose of providing universal text explanation.

Minimal Permission Use

  • We use activeTab permission where possible
  • Content scripts only inject minimal UI elements (a button and tooltip)
  • No background data collection or persistent tracking
  • Extension only activates when you select text

Privacy Policy

Overview

Ask Gemini is committed to protecting your privacy. This policy explains what data we collect, how we use it, and your rights regarding your information.

Data Collection

Ask Gemini collects and stores the following data locally on your device:

  • API Key: Your Google Gemini API key, stored in Chrome's sync storage to enable API requests.
  • User Preferences: Your custom question template settings.

We do not collect:

  • Personal information
  • Browsing history
  • Analytics or usage data
  • Any data on external servers

Data Usage

When you highlight text and click the Ask Gemini button:

  • The selected text is sent directly to Google's Gemini API using your API key.
  • The request is made from your browser to Google's servers.
  • No data passes through any third-party or intermediary servers owned by us.

Data Storage

  • All settings are stored locally using Chrome's storage.sync API.
  • Data syncs across your Chrome browsers if you're signed into Chrome.
  • No data is stored on external servers controlled by us.

Third-Party Services

This extension uses the Google Gemini API. When you make a request, your highlighted text is sent to Google. Please review Google's Privacy Policy for information on how they handle data.

Data Security

  • Your API key is stored in Chrome's built-in storage system.
  • All API requests are made over HTTPS.
  • We recommend not sharing your API key with others.

Your Rights

You can:

  • Delete your stored API key and settings at any time through the extension popup.
  • Uninstall the extension to remove all locally stored data.
  • Clear your API key from Chrome's storage via Chrome settings.

Children's Privacy

This extension is not intended for children under 13. We do not knowingly collect data from children.

Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected on this page with an updated revision date.

Contact

For questions about this privacy policy or the extension, please open an issue on our GitHub repository.
